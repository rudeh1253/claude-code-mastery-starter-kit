## 환영합니다 👋

이 블로그는 **데이터베이스 없이** 소스 코드에 포스트를 작성하고,
빌드 타임에 정적으로 생성(SSG)됩니다.

새 글을 쓰려면 `content/posts/` 디렉터리에 `.mdx` 파일을 추가하기만 하면 됩니다.

### 코드 블록 예시

```ts
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("world"));
```
